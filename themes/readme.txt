# MWR 2026-05-21

Structure à respecter : une thématique = un fichier .qmd à la racine de ce sous-dossier (p.ex. 'energie.qmd')
Le sous-dossier correspondant où sont stockés les fiches .html à afficher doit être nommé pareil (energie/).

Uniquement les sous-dossiers dont le nom correspond à un .qmd sont utilisés dans le site quarto
